## Estructura del repositorio e instrucciones de ejecución

### Estructura del repositorio

Este repositorio contiene a los archivos fuente dpllSolver.cpp, coloringSolver.cpp y el archivo de cabecera DPLL.h, que es necesario para la correcta compilación de ambos archivos. Hay un archivo makefile que contiene instrucciones de compilación y limpieza de los ejecutables.


### Instrucciones de ejecución
  Compila los archivos necesarios para tener los ejecutables dpllSolver y coloringSolver. También puede usar alternativamente los siguientes comandos para tener uno de los ejecutables o el otro
```bash
  make
```
```bash
  make compile_dpll
  ```
```bash
  make compile_coloring
```
  Elimina los ejecutables generados
```bash
  make clean
  ```
